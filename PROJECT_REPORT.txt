1. PROJECT OVERVIEW
--------------------------------------------------------------------------------
Mr. Nila Tex is a premium clothing and textile showroom website based in 
Addalaichenai, Sri Lanka. The website showcases products for men, women, 
kids, and offers quality fabrics and textiles. Features include product 
browsing, category filtering, search functionality, and direct WhatsApp 
ordering.

7. DISCOUNT SYSTEM LOGIC
--------------------------------------------------------------------------------
The discount system uses a priority-based approach:

Priority Order:
1. GLOBAL_DISCOUNT (from CONFIG) - Applied to ALL products if set > 0
2. Calculated Discount - Calculated from Old Price + Price when Old Price exists
3. Sheet Discount - Value from 'Discount %' column (used only if no Old Price)
4. No Discount - Default when neither is provided

Discount Calculation Formula:
    calculatedDiscount = (1 - Price / OldPrice) × 100

Badge Colors by Discount Level:
- SUPER (≥50%): Green (#27ae60)
- HIGH (≥30%): Red (#e74c3c)
- MEDIUM (≥15%): Orange (#e67e22)
- Default (<15%): Gold (#c9a959)

8. SECURITY IMPLEMENTATIONS
--------------------------------------------------------------------------------
XSS Protection:
- escapeHtml() function for all user-generated content
- Applied to: product names, brands, badges, item codes, descriptions
- WhatsApp messages use escaped content
- Form submissions use escaped user input

Form Validation:
- Email format validation (regex)
- Phone number validation (regex)
- Required field validation
- Focus on invalid field on error

9. IMAGE HANDLING SYSTEM
--------------------------------------------------------------------------------
Smart Image Display Logic (getProductImageHTML function):

1. Both images exist & different → Flip effect enabled
2. Only front image → Single image (no flip effect)
3. No images or load error → Placeholder SVG

Key Features:
- Automatic fallback to placeholder on image load failure
- Consistent aspect ratio (4:5) maintained
- Lazy loading for performance
- onerror handler for runtime image errors

## 13. DESIGN SPECIFICATIONS

### 13.1 Concept & Vision
Mr. Nila Tex is a distinguished clothing and textile establishment in Addalaichenai, 
Sri Lanka, offering premium fashion for the entire family. The website embodies 
**luxury without pretension** - a digital flagship store that feels like walking 
into an upscale boutique with warm Sri Lankan hospitality.

### 13.2 Design Language

#### Aesthetic Direction
**Elegant Minimalism meets Sri Lankan Warmth** - Clean lines, generous whitespace, 
and sophisticated photography create a gallery-like experience, while gold accents 
and subtle textures add warmth and premium feel.

#### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Black | #1a1a1a | Sophisticated, authoritative |
| Pure White | #ffffff | Clean canvas |
| Cream | #f8f6f3 | Warm background alternative |
| Gold Accent | #c9a959 | Premium, luxurious touch |
| Soft Gold | #e8dcc8 | Subtle highlights |
| Dark Gray | #4a4a4a | Secondary text |
| Light Gray | #9a9a9a | Muted text |

#### Typography
| Element | Font | Style |
|---------|------|-------|
| Headlines | Playfair Display (serif) | Elegant, editorial feel |
| Body & UI | Poppins (sans-serif) | Modern, readable, friendly |
| Accents | Cormorant Garamond | Special quotes/taglines |

#### Spatial System
- Base unit: 8px
- Section padding: 80px-120px vertical
- Card gaps: 24px
- Maximum content width: 1400px
- Mobile padding: 20px

#### Motion Philosophy
| Animation Type | Duration | Effect |
|----------------|----------|--------|
| Entrance animations | 400ms | Fade-up with stagger (ease-out) |
| Scroll animations | - | Subtle parallax, reveal on intersection |
| Hover states | 300ms | Scale 1.02-1.05, shadow lift, color transitions |
| Product cards | 600ms | Image zoom on hover, overlay reveal |

#### Visual Assets
- Icons: Lucide Icons (line style, 1.5px stroke)
- Images: Unsplash fashion/textile photography (high-quality, editorial)
- Decorative: Subtle grain texture overlay, geometric gold accents
- Backgrounds: Soft gradients, subtle patterns
