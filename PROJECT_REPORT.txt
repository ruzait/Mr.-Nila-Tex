1. PROJECT OVERVIEW
--------------------------------------------------------------------------------
Mr. Nila Tex is a premium clothing and textile showroom website based in 
Addalaichenai, Sri Lanka. The website showcases products for men, women, 
kids, and offers quality fabrics and textiles. Features include product 
browsing, category filtering, search functionality, and direct WhatsApp 
ordering.

Website URL: https://mr-nilatex.netlify.app

2. WEBSITE STRUCTURE
--------------------------------------------------------------------------------
Pages:
- index.html (Home Page)
  - Hero section with slider
  - Features bar
  - Categories section
  - Products showcase (all products)
  - Offers/Ramadan sale section
  - About section
  - Testimonials
  - Contact section with form & map
  - Footer

- collections.html (Collections Page - NEW)
  - Page header with breadcrumb
  - Products showcase with Load More pagination
  - Category filters
  - Search functionality
  - WhatsApp ordering per product

3. TECHNOLOGIES USED
--------------------------------------------------------------------------------
Frontend:
- HTML5 - Semantic markup
- CSS3 - Styling with CSS Variables, Grid, Flexbox, Animations
- JavaScript (ES6+) - Interactive functionality
- Font Awesome 6.5.1 - Icon library
- Google Fonts - Playfair Display & Poppins
- AOS Animation Library 2.3.1 - Scroll animations

Data & Backend:
- Google Sheets - Product data storage (CSV export)
- PapaParse 5.4.1 - CSV parsing
- LocalStorage - Client-side caching (5 minutes TTL)

External Services:
- Google Maps Embed API - Location display
- Netlify - Website hosting
- WhatsApp API - Direct messaging for orders
